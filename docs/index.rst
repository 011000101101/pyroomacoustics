Welcome to pyroomacoustics's documentation!
===========================================

Pyroomacoustics is a package for audio signal processing in an indoor environment. It has the following features.

* Image source model for the generation of room impulse responses.
* Short Time Fourier Transform (STFT) engine for fast processing of audio signals.
* Spatialized sound source and receiver classes.
* Beamformer class to quickly prototype beamforming algorithms.
* Many other useful routines.
